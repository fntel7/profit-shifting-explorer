# SEO & discoverability checklist

High-leverage actions to make the profit-shifting research easier to find.
Do not add citation counts, rankings, or h-index anywhere on the site.

## On-site (Google Sites)

- [ ] **Page title (browser tab).** Set to: `Fotis Delis — Economist, Profit Shifting & Corporate Taxation`.
- [ ] **Hero subtitle** (acts as meta description for search):
      *Economist at the European Commission Joint Research Centre (JRC). Research on profit shifting of multinational enterprises, international corporate taxation, and tax avoidance.*
- [ ] **Keywords in body text** (embed naturally, do not keyword-stuff):
      "profit shifting", "multinational enterprises", "corporate tax avoidance", "MNE tax planning",
      "international corporate taxation", "Orbis firm-level data", "tax havens", "BEPS", "Pillar Two".
- [ ] **Descriptive link anchors.** e.g. *"Download the country-year profit-shifting dataset"* (not "click here").
- [ ] **Icon-link row** in the hero block links out to every external profile (see homepage.md).

## Off-site profiles (one-time cleanup)

- [ ] **Google Scholar** — add website URL in profile; verify all papers are claimed; add a photo.
- [ ] **ORCID** — fill in completely; link to website and to each paper DOI.
- [ ] **SSRN author page** — add affiliation, photo, link to website.
- [ ] **RePEc author service** — register and claim all papers. Consolidate the
      variants ("Delis, Fotios" / "Fotis Delis" / "DELIS Fotis") under a single author ID.
- [ ] **CEPR profile** — add website link if missing.
- [ ] **JRC profile** — add website link if missing.
- [ ] **LinkedIn** — add the website in the "Featured" section; post when new papers drop.
- [ ] **BlueSky / X** *(optional but high-leverage for the econ/tax community)*.
- [ ] **Google Search Console** — submit the `sites.google.com` URL; verify ownership.
- [ ] **ResearchGate** — claim profile, upload accepted manuscripts where copyright allows.

## Citation-driving actions (the real lever)

- [ ] Write a **VoxEU-style companion piece** for *The Pricing of Profit Shifting* after peer review.
- [ ] Deposit the public country-year dataset on **Zenodo** — gets a DOI, increases discoverability.
- [ ] Present at **NBER Public Finance**, **NTA Annual Conference**, **IIPF** — not only European venues.
- [ ] Tag co-authors on social posts; cross-posting multiplies reach.

## What NOT to do

- No live citation counter on the website.
- No Google Analytics or other trackers.
- No ABS-star or JCR-rank annotations inline in website body text (keep those in the CV only).
