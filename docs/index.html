How to read the map.
  1. Select a GUO country and a year.
  2. The map shows lowest-tax jurisdictions linked to MNE groups whose parent firm is in the selected GUO country.
  3. Shaded countries with bubbles are those lowest-tax jurisdictions. Bubble size reflects the estimated profit shifted for that GUO × lowest-tax country combination. Values are group-level aggregates, not bilateral profit flows.
  4. The selected GUO country itself appears unshaded and without a bubble.
  5. Light grey countries have no estimate for the selected combination.
